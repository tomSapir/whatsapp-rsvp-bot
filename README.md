# WhatsApp Guest Confirmation Bot

A WhatsApp chatbot that confirms guest arrival for an event.

> ⚠️ **Planning in progress.** Scope, tech stack, and architecture are being defined.
> This README will be filled in once the plan is finalized.

## Goal

Automatically reach out to invited guests over WhatsApp and collect/confirm
their arrival (RSVP), tracking responses centrally.
